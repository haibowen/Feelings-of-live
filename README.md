# 2019这一年

# 实习那些事
从考完最后一门考试课，到离开最后一晚的宿舍，从大一的满心欢喜，到大四的离校实习，所有的一切看起来好像都是在按部就班，所有的一切看起来都似乎井然有序。2018年8月4号，那是我刚踏入北京城的时候，我发现一切并不是自己想像的那么所以然，这是我第一次来北京，我一度感到无所是从，莫名的恐惧感油然而生。我一边在忙着复习自己的技术知识点，一边在忙着约面试，我感叹八月天气的炎热，感叹生活的不易，可是转念一想，踏入社会哪有容易的事，就这样，我从一个习惯了在学校里生活的学生，忙不迭的进入了社会这个浪潮。
    从我开始学习编程的时候，身边就有好多人在感叹，感叹编程的种种好处，感叹着程序员随随便便就能在大城市找到一份较为满意的工作，感叹着大城市的生活如何如何精彩，诚然上几届的学长也都是这般，他们为我们树立了榜样，用他们的实践向我们讲述了事情地的确是这样。于此，我也一度对自己的前途感到满意，心想将来一定会出人头地。我加入了科研室，我在学校有了事情可干，我也不再每天都泡图书馆，看一些不痛不痒的文字了，日子在这样不经不意之间流转。当我从cmd命令窗口，发出来自java的第一句问候的时候，我知道我自己的编程生涯开始了，我在不断的跌倒，我在不断的奔跑，来自ide的鲜红的error一次次的考验着我的耐心，一次次的嘲讽着我拙劣的编程技术，我一边感叹自己的一无所获，一边又将编程基础翻了几遍，期盼着能够实现自我救赎。就这样，我坚持了大概两年时间吧，算是还有一点小小的收获。
夏末来到北京，开始了自己的找工作的生活，我打印了一沓简历，借住在表哥那里，表哥待我蛮好，细心的为我讲解了好多知识要点，虽然我以前一直觉得表哥很厉害，但是我并不觉得他在IT上有多厉害，心想自己毕竟是专业的嘛，可是一天晚上，表哥为我讲解一个知识点的时候，我对表哥开始佩服的五体投地，表哥有一个书架，上面放着他自学看的书，书并不是特别多，只是我三年读的技术书加上自己的专业课本都没有这些书多，可想自己的水平到底怎样。幸运的是我投了几家公司，面了三家，有两家要录用我，那是候自己因为还有学校的比赛要参加，所以我就选择了一家自己觉得能够学习技术的公司，开始了自己的实习生涯，从开始找工作，到我正式入职，花了一周的时间，随后我的实习生涯就正式开始了。
当我开始工作的时候，无所事事，已然不复存在。第一周工作，忙的我晕头转向，leader对我的要求就是，修改项目上的bug，无奈在熟悉完了工作要求后，便投入到与bug的争斗中，经过自己的一周摸索，一番战斗，我    光荣的在一周时间内修改出了一个bug，当时自己的心态已然开始崩溃，就差最后的彻底爆发，好在leader特别照顾我，一边安慰我，一边在给我强调，不要着急，静下心来，把代码看明白了，努力学习就好，不要心急。我在这里对她真诚的说声感谢！接下来的一周，就顺畅多了，我自己改bug的速度有了质的提升，自己一周居然可以修改出三个bug了，我对自己的进步也有点满意，随后的日子里，我改bug基本越来越顺，我名下的bug很快就被我修改完了，我一边暗暗欣喜，一边心想bug不过如此嘛，我开始学习了一些基本的编程基础，懂得了什么叫服务器编程，了解了make file 了解了版本控制工具的使用，就这样第一阶段的改bug任务基本结束了。
  有一天，我闲来无事，在翻看自己之前做的修改，看着pc上提示有邮件，我点开看了一下是leader来的邮件，说客户有了新的需求，我们要开始做新的需求了，leader给我分配了几个新的需求，让我评估一下工期，尽快给她回复邮件，我看了几眼，发现自己的理解，不是很到位，心想leader是不是搞错了，要给我派发新的需求，我的能力可以吗？她是不是发错了，我当即就去请教了她一下，她也耐心的解答了我的困惑，没错邮件就是发给我的，在她的指导下，我把工期定了下来是九天，这之后便开始了我的新需求阶段，接到新需求以后，我变得开始紧张起来，不确定自己能不能做出来，当然这个阶段还是少不了自我安慰还有leader的帮助，最终自己还是在规定日期内，把新的需求做了出来，做完了新需求感觉自己如释重负，感觉做需求也不过如此。在我还没有休息够的时候，新的需求又来了，好吧，我就陷入了无限的做新需求的过程中。好在这样的日子并没有持续了特别长的时间，我在11月份，拥有了自己一个月的基本空闲的时间，我制定了一下基本计划，把安卓艺术开发探索，还有第一行代码，趁机复习了一遍，又把java基础给过了一遍，怎么说收获还是蛮大的，毕竟古人常言温故知新嘛，自从看了遍书以后，发现自己对一些编程的设计还有一些问题有了自己的见解，对一些leader提出的一些基本问题，能够很快找到对策，工作效率也开始有了提高。复习闲暇的日子，总是那么短暂，进入12月，项目上又开始有了新的需求，所以我开始了做自己的新的需求，一切还是按部就班，就像之前一样，唯一的大变化就是，自己看到问题不在变得束手无策，而是开始努力寻找解决之道，开始多个角度体会思索问题的解决方案。这获许可能就是内在的提高吧，当你付出了努力，一切的结果都是水到渠成，没有那么想当然，但也会是特别的自然。
   写这篇文章的时候已经过了冬至，自己的实习生涯也行将结束，有时候会想，夏至炎热面试失败后的路上，那颗止不住凉意的心，此刻或许已经开始在隆冬的大街上，逐渐变得温暖起来了吧！我期待着春风的吹起，期待着漫山的花香，期待着能有所获，感悟至此，奋斗不已！

# 2020年3月

我努力的把思绪拉回到这个月的月初，发现20多天前的日子，竟没有了丝毫印记，也许是每一天的平淡，都没有在脑海里留下特别值得铭记的回路，我时常在想，如何才能让某一天中的某一时，某一刻，显得有某些意义？这个问题我每次都会问自己，但是我直到今天我还是说不出一个所以然。我还在思考一个这样的问题，什么样的事才会在自己身上留下印记？我总会在刷完抖音以后，感到一丝丝的罪恶感，这或许就是人的惰性吧，骨子里都是喜欢轻松的，舒服的，毕竟刷抖音不需要任何的思索的！三月份又翻看了周杰伦的头文字D和不能说的秘密，发现自己还是喜欢一些有时代感的画面，如果不能说的秘密会复映的话，我真的会选择去影院看一次，还有那应景的歌词，最美的不是下雨天，是和你一起躲过雨的屋檐！想必桂纶镁是那个时代好多人的女神吧！<br><br>

总之四月也算是今年过得不太好的一个月份了，现在思绪有点乱，想到哪里，就写到哪了！还好，就这样吧先！
# 2020年5月
今天是2020年6月5号 但这篇是五月的月记，五月工作上着实有点忙，月记也没来得及写，因为我月底还在加班，拖到今天终于有空补一补了。五月也是一个压抑的一月，好多之前自己没做过的工作，不会的都要硬着头皮顶上，没办法社会就是这样子的，不会给你太多理由去讲为什么，站在资本家的角度来看，我招你就是来干活的，所以他们不会考虑是否对你职业生涯有利，只要能完成工作的才是合格的员工。这个月也在硬着头皮跟很多之前不认识的同事在交流请教，因为不问，不请教，工作是没法开展的，各种滋味，只有自己体会，还好熬过了难过的五月！
这一月，自己的课余时间不多，精神文明建设也很少，没有继续读书，没有开展其它的活动，床头的kindle一如既往的显示着黑色的屏保，桌上的台灯陪我到夜晚的深处，电脑开了关，关了开，闪过的每一帧开机画面，都在倒映这个眉头紧锁的少年！关灯辗转反侧的夜晚，早上如期的闹钟，匆忙的地铁人流，一幕接着一幕的画面都在记录着这一月的每一分每一秒，此刻或穷或富，或甜或苦，或迷茫或投入，难以言说！
几年后，回想起来这段焦头烂额的日子又会是怎样，好像是生活在给你开玩笑，而你能做的就是 即使生活给了你一百种哭泣的理由，你也要学会以一千种理由去微笑，毕竟有些事，谁都替代不了谁，可能当时难过的无法呼吸，但是走过去就会发现，那些没有难倒你的，终将使你变得强大！
